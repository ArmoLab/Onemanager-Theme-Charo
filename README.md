## Live site

Prebuilt [Demo](https://charo.koto.cc/)



## To devlop

Need node.js 16+ and pnpm

```bash
pnpm i
pnpm run dev
```

## To build

Need node.js 16+ and pnpm

```bash
pnpm i
# to preview
pnpm run build
# to prod
pnpm run build:deploy Domain=<your custom domain>
```
